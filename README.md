# Henrotte VPS

---

## Install Server

+ ssh-keygen -R henrotte.fr

+ sudo passwd root

+ sudo apt-get install git

+ git clone https://github.com/YvesCousteau/henrotte_c..git

+ sudo apt-get install npm

+ npm i

+ npm i nodemon

+ npm i pm2 -g

+ pm2 start app.js

+ sudo apt-get install certbot

+ certbot certonly --standalone -d henrotte.fr
